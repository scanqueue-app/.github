# ScanQueue

**QR-based queue management for barbershops, salons, clinics, restaurants, and events.**

Turn chaotic walk-in lines into live, text-notified waitlists. Customers scan a QR code at the door, see their spot live, and get SMS or WhatsApp when it's their turn. No apps, no hardware, no training.

### Stack
Next.js 16 · React 19 · TypeScript · Supabase (Postgres + Realtime + Auth) · Tailwind CSS 4 · Vercel

### Features
- QR check-in — customers join in 10 seconds
- Live position + wait-time tracking on the customer's phone
- SMS, WhatsApp Business, voice notifications, AI receptionist
- Multi-counter dashboard with shared "Next Available" pool
- Appointment scheduling alongside walk-ins
- Multi-location / enterprise support
